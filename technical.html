Research

I use Python (numpy, matplotlib, scipy) and Jupyter notebooks daily for research. Some examples of my technical expertise include:

  • Processing 10's of TBs of astronomical data using Grid Engine for multi-node processing
  • Developing scripts and source code in Python for the analysis of data
  • Regular usage of Git, Unix, and LaTeX
  • Applying linear algebra (i.e., covariance weighting), mathematical (i.e., Fourier analysis and filtering), and statistical (i.e., linear regression, bootstrapping, Bayesian statistics) methods
  • Adapting code for parallel computing using MPI for Python
  • Developing content for Jupyter notebook tutorials for teaching/mentoring purposes

Web Application

My research group has weekly Friday lunches where we bond over free food and share research updates. Every week, a different member of the group volunteers to order lunch, and we used to track this rotation on a whiteboard with tally-marks next to names. As a side project, I developed a web application using Flask, SQL, HTML, CSS, and Python which allows users to sign-up to host lunch for a specific date and tracks how many lunches each person has bought. The application can be found here. It is deployed on Google Cloud using Docker and Kubernetes.
